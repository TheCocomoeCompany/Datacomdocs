title: FileZilla
---

**自由且跨平台的FTP软件**

## 特点

- 可以**断点续传**进行上传、下载（需要服务器支持）
- 自定义命令
- 可进行站点管理
- 防发呆功能（有的FTP服务器会将发呆过久的用户赶出，这样发呆的用户就得重复登录）
- 超时侦测
- 支持**防火墙**
- 支持**SOCKS4/5**、**HTTP1.1**代理
- 可进行**SSL**加密连接
- 支持**SFTP**（Secure FTP）
- 可以排队进行上传、下载
- 支持拖放
- 多国语言支持，包括**简体**、**繁体中文**
- 可通过**Kerberos**进行**GSS**验证与加密

## 获取FileZilla

:::tip

FileZilla是一款免费的FTP工具包，包含服务端与客户端。

您可以在[FileZilla中文网](https://www.filezilla.cn/)获取。

:::
